Rendering HTML in React

  • The HTML code must be wrapped in ONE top level element.
    • So if you like to write two paragraphs, you must put them inside a parent element, like a`div`element.
  • you can use a fragment instead of a div element .
  • A fragment looks like an empty HTML tag:`<>`.
  • elements should be closed.
  • Attribute class = className
  • The `class` attribute is a much used attribute in HTML, but since JSX is rendered as JavaScript, and the `class` keyword is a reserved word in JavaScript, you are not allowed to use it in JSX.
  • Note that in order to embed a JavaScript expression inside JSX, the JavaScript must be wrapped with curly braces,`{}`.
  • React requires your function name to begin with a capital letter.
